# Стабилизация перевернутого маятника на тележке

## Обзор
В данном проекте реализован синтез системы управления для перевернутого маятника на тележке с использованием нескольких методов: операторный метод (передаточные функции) и методы пространства состояний (размещение полюсов, проектирование наблюдателя).

## Методы и инструменты
- Математическое моделирование: уравнения Лагранжа, линеаризация, представление в пространстве состояний
- Синтез регулятора: размещение полюсов, LQR, операторный метод
- Проектирование наблюдателя: наблюдатель полного состояния с размещением полюсов
- Анализ устойчивости: карты полюсов-нулей, анализ собственных значений
- Моделирование: Python (scipy, numpy, control, matplotlib, pygame)
- 
## Ключевые результаты
- Разработаны два регулятора: на основе операторного метода и метода пространства состояний
- Синтезирован наблюдатель полного состояния для систем с неполным измерением состояния
- Реализована интерактивная анимация с использованием pygame
- Проведено сравнение качества работы при различных начальных условиях (0.02, 0.05, 0.3 рад)
